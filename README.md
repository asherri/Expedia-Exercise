**SETUP**
* `$ git clone https://github.com/asherri/Expedia-Exercise.git`
* `$ cd Expedia-Exercise`
* `$ mvn clean package`
* `$ mvn package`
* `$ mvn compile war:exploded`

**Assumptions**
* You can send any type of parameters: string, integer, or double
**known issue**
* I did not face any problems so far in my example.
